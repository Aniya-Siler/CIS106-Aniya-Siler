This program asks user for full name and displays it as lastname, first initial

Main 
    get full name
    get first initial
    get last name
    display name

Get full name:
    Parameters:
        None
    Process:
        "What's your full name?"
        Get full name
    Return Value:
        full name

Get first initial:
    Parameters:
        full name
    Process:
        Split full name
        Get firstname
        Get first initial
    Return Value:
        first initial
        
Get lastname:
    Parameters:
        full name
    Process:
        Split full name
        Get lastname
    Return Value:
        lastname
         
Display name:
    Parameters:
        first initial
        lastname
    Process:
        display last name and first initial
    Return Value:
        None
