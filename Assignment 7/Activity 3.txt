This program asks the user to select US (miles, yards, feet, and inches) or Metric (kilometers, meters, and centimeters) and input a given measurment. Then the program converts the given measurement and displays the result.

Main Program
    Get Choice
    Either:
         Get US measurment
         Calculate Metric
         Display Result
Or:
    Get Metric measurment
    Calculate US
    Display Result
    
    
Get Choice
    Parameters:
    None
    Process:
    Display prompt
    Get choice of Us or Metric conversion
    Return Value:
    Choice
    
    
Get Tempature
    Parameters:
        Label(US or Metric)
    Process:
        Display prompt with label
        Get Measurment
    Return Value:
        Measurment
        
Calculate US
    Parameters:
        Metric measurment
    Process:
        Convert Metric measurment to US
    Return Value:
    US measurment
    
Calculate Metric
    Parameters:
        US measurment
    Process:
        Convert US measurment to Metric
    Return Value:
        Metric measurment
        
        
Display Result
    Parameters:
        Input measurment
        From label (US or Metric)
        Output measurment
    Process:
        Display measurment and labels
    Return Value:
        None
