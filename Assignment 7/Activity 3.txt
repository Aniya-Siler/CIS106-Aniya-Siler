This program asks the user to select miles, yards, feet, and inches and input a given measurment. Then the program converts the given measurement and displays the result.

Main Program
    Get Miles
    Get Choice

    Either:
        Calculate Yards
        Calculate Feet
        Calculate Inches
        Display US
    Or:
        Calculate Kilometers
        Calculate Meters
        Calculate Centimeters
        Display metric
    Or:
        Display error message

Get Choice
    Parameters:
        None
    Process:
        Display Prompt
        Get Choice of US or metric
    Return Value:
        Choice

Get Miles
    Parameters:
        None
    Process:
        Display Prompt
        Get Miles
    Return Value:
        Miles

Calculate Yards
    Parameters:
        Miles
    Process:
        Convert Miles to Yards
    Return Value:
        Yards

Calculate Feet
    Parameters:
        Miles
    Process:
        Convert Miles to Feet
    Return Value:
        Feet

Calculate Inches
    Parameters:
        Miles
    Process:    
        Convert Miles to Inches
    Return Value:
        Inches

Calculate Kilometers
    Parameters:
        Miles
    Process:    
        Convert Miles to Kilometers
    Return Value:
        Kilometers

Calculate Meters
    Parameters:
        Miles
    Process:    
        Convert Miles to Meters
    Return Value:
        Meters

Calculate Centimeters
    Parameters:
        Miles
    Process:    
        Convert Miles to Centimeters
    Return Value:
        Centimeters

Display US
    Parameters:
        Miles
        Yards
        Feet
        Inches
    Process:
        Display Measurments and labels
    Return Value:
        None

Display Metric
    Parameters:
        Miles
        Kilometers
        Meters
        Centimeters
    Process:
        Display Measurments and labels
    Return Value:
        None
