# This program uses a loop to generate a list of
# multiplication expressions for a given value.

Main Progam
    Loop
        Get value
        Get expression count
        Process expressions
        Get answer
        While answer is yes

Get answer
    Parameters:
       Name
    Process:
       Display prompt (continue or not)
       Get answer
    Return Value:
       Answer

Get Value
    Parameters:
       Name
    Process:
       Display prompt
       Get Value
    Return Value:
       Value
 
Process Expressions
    Parameters:
        Value
        Expression count
    Process:
     Initialize count
        Loop
            Display count
            expression count
            While count <= expression count
     Return Value:
        None

Display Expression
    Parameters:
        Value
        Count
        Result
    Process:
        Display expression
    Return Value:
        None
